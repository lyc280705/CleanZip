Install

Download the package and start from Finder.

  1. Download CleanZip-2.6.29.pkg.
  2. Open the installer. It installs CleanZip.app and the Finder service.
  3. Select files, folders, or archives in Finder and choose CleanZip Compress or Extract.

FAQ

A few details before you download.

Does CleanZip upload files?

No. Operations run locally on your Mac with system tools and bundled 7zz.

Can CleanZip create RAR archives?

No. CleanZip can extract RAR archives, but archive creation is intentionally focused on ZIP and 7Z.

Is the package notarized?

No. The package is ad-hoc signed for local distribution, but it is not notarized with an Apple Developer ID.
